clean(mvn clean) - удаляет результат предыдущей сборки типично clean package

validate - быстрая проверка, что проект валидный, есть валидный pom и базовая структура

compile - компиляция src/main/java --> target/classes быстрая проверка компиляции кода

test - запускает unit-тесты типично перед package (mvn clean package -DskipTests - позволяет пропустить тесты про сборке)

package - собирает артефакт проекта и кладет его в target

verify - проверка качества сборки, валидация артефактов, плагины контроля

install - кладет артефакт проекта в локальный репозиторий

site - генерация документации и отчетов

deploy - публикует артефакт проекта на удаленном репозитории, делает релизную версию. Альтернатива: Jenkins, Tekton
