GET
Для чего: получение данных
Safe: да
Идемпотентность: да, повтор дает один и тот же результат
Кэшируемость: да
Не использовать: для создания или изменения данных

POST
Для чего: что бы создать новый ресурс
Тело запроса: JSON
Safe: нет
Идемпотентность: нет, повтор запроса создаст дубликат
Кэшируемость: нет
Не использовать: для получения данных

PUT
Для чего: полная замена ресурса
Тело запроса: JSON
Safe: нет
Идемпотентность: да
Кэшируемость: нет
Не использовать: если меняется только часть полей или нет возможности отправить объект целиком

PATCH
Для чего: частичное обновление ресурса
Тело запроса: JSON
Safe: нет
Идемпотентность: да
Кэшируемость: нет

DELETE
Для чего: удаление ресурса
Тело запроса: нет
Safe: нет
Идемпотентность: да
Кэшируемость: нет
Не использовать: если запрещено физическое удаление данных

HTTP-статусы:
200 OK успешное чтение или обновление ресурса
201 Created успешное создание ресурса (POST)
204 No Content успешное обновление без тела

400 Bad Request невалидные данные (валидация DTO)
404 Not Fount ресурс не найден
409 Conflict конфликт состояний

500 Internal Server Error - неожиданная ошибка


GET: /api/cars выдает список автомобилей
GET: /api/cars/11 выдает авто по id
