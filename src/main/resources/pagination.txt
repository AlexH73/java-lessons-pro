Pageable объект, который описывает как читать/показывать страницу с данными.
page - номер страницы, начиная с 0
size - количество элементов на странице
sort - сортировка (поле + направление), например sort=price,asc или sort=price,desc

Page<CarDto> - объект, который содержит
список автомобилей на странице
общее количество элементов (totalElements)
общее количество страниц (totalPages)
флаги (first, last)
